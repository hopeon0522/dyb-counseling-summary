설정

현재 GitHub Pages/PWA에서는 보안상 메일을 사용자 확인 없이 자동 발송할 수 없어 메일 앱 작성창을 엽니다. 완전 자동 발송은 별도 백엔드나 이메일 API 연결이 필요합니다.

참고 예시 평균 분량 저장된 예시 없음

참고 예시가 없으면 상담 내용에 맞는 자연스러운 분량으로 요약합니다.

Gemini API 키는 이 기기의 localStorage에만 저장됩니다. 공용 기기에서는 저장하지 마세요.
